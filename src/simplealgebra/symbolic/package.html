simplealgebra.symbolic


In Electrostatics, Gauss' Law (see http://en.wikipedia.org/wiki/Gauss's_law) could potentially be written as:

∇ • E x , y , z = ρ x , y , z ϵ 0

where E and ρ are functions with explicit parameters x, y, and z. However, Gauss' Law is instead typically written as:

∇ • E = ρ ϵ 0

where x, y, and z are understood to be implicit parameters of the functions E and ρ. In SimpleAlgebra, functions like E and ρ above are represented using class simplealgebra.symbolic.SymbolicElem. In cases where the enclosed type of a SymbolicElem is another SymbolicElem, it follows that the enclosing SymbolicElem represents a function of functions. In SimpleAlgebra, all functional parameters are implicit.

The DISTRIBUTE_SIMPLIFY and DISTRIBUTE_SIMPLIFY2 operations provide the ability to reduce a SymbolicElem function to a simpler SymbolicElem function based on a set of rules. The simplification rules are configured using a set of Drools DRL files. It is possible to reconfigure the DRL simplification rules without recompiling the SimpleAlgebra Java code.

In functional languages such as Lisp and Scheme, it is possible to create functions that generate other functions. A function of functions in SimpleAlgebra can be considered an alternate representation of a function that generates another function. That is to say, a SimpleAlgebra function of functions is a way to express a λ calculus.
